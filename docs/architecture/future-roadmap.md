# Future Roadmap

**Project:** Atlas  
**Portfolio:** WolfSec Labs

Atlas is the first repository in the WolfSec Labs portfolio.

Its purpose is to establish a realistic enterprise environment that will be reused throughout every future project.

Rather than building isolated demonstrations, every repository expands the same fictional company and infrastructure.

---

## Sentinel

Focus

Endpoint monitoring and detection engineering.

Atlas provides the infrastructure that Sentinel will monitor.

---

## Hunt

Focus

Enterprise security assessment and attack simulation.

Hunt evaluates the security posture of the environment created in Atlas.

---

## Forge

Focus

Security hardening and remediation.

The weaknesses identified during Hunt become the improvements implemented in Forge.

---

## Long-Term Vision

By the end of the portfolio, WolfSec Labs will represent a complete security engineering lifecycle.

Infrastructure is designed in Atlas.

Visibility is introduced in Sentinel.

Security is assessed in Hunt.

Controls are improved in Forge.

Each repository builds on the previous one to tell a single engineering story rather than presenting unrelated cybersecurity projects.
